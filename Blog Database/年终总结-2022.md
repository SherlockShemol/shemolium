# 年终总结-2022

2022就要结束了，一开始打算写一个学期总结，想了一下，决定干脆写一个年终总结来总结这一年。

# 心态
这一年的心理变化十分巨大，上半年自暴自弃，下半年找回希望改变自己。不过我不想回溯了，一旦过了一个节点，我就会把之前所有事情抛之脑后，我本想起码写出那把促使我改变的钥匙，但是又想了想，拿起那把钥匙的动机似乎早已存在在我的思考中，不可追溯。经历了一些黑暗时刻，所幸我已经爬出谷底，心态上更加开放，更愿意接受新鲜的事物，记得刚上大学的时候，既怕困难也害怕新鲜的事情，遇到困难总有一种逃避的冲动。当感觉自己步步维艰时，就是必须要改变自己的时候了。

# 学业
上半年在逃避，下半年直面困难。虽然做的还不尽完美，还有很大可以完善的空间，但我相信自己是越来越好的，明年的主题应该就是考研了，希望可以在这一年里扎实打下学科基础，并且不断改掉自己的坏习惯。要更加专注，要付诸更多思考，要全情投入。

# 社交
下半学期和人的相处更加大胆了，会主动去接触一些人并和他们交流，随着自己和越来越多的人交流相处，感觉到自己心智越来越成熟，而且在对人际关系的处理上更加得体。在和别人接触的时候，自己也会反思有时候哪里做的不对，有些时候可能太过热情，有些时候可能太过冷漠，有些时候没管理好自己的情绪，这样想想就觉得，哎呀，感谢大家这么宽容，我会继续改进的。 脸皮也变得厚了，自己写邮件向论文作者咨询代码，咨询研究生招生的相关情况，向学长咨询实验室的情况等等。推己及人，我觉得只要保持真诚礼貌，保持感谢，然后大大方方向对方咨询就可以了。如果被拒绝，也不用灰心，因为不是每个人都有义务回答你的疑问。道谢之后再想别的办法就是了。

# 情感
还没有进展，但感觉到自己和一年前相比更加成熟也更加理性。就这样不断成长，相遇的人终会相遇。

# 生活
这一年有什么自己感兴趣的东西，就会去尝试，成为一个参与者（Stable Diffusion，ChatGPT，Linux）而不是只是成为一个旁观者，然后去体会自己在面对这些事情时的真实感受。总的来说，玩这些让我很开心。广泛涉猎后下一步其实就是把时间和精力集中在一个点上了。Focused and determined. 更加体谅家人，理解家人，爱家人了。感谢家人在我最糟糕的时候还能无私地爱我，我也不会辜负他们。 外部环境 今年社会上发生了很多不可思议的事情，我已经不知道该说些什么了，只是告诉自己别忘记，然后专注在自己正在做的事情，这样才能改变自己的处境，照顾好家人。

# 阅读
今年读了20本左右，差不多是去年的一半。主要都在寒暑假读书，上半年期中那会读了大半本张维迎的《经济学原理》，然后下半年期中考试完读了Linus自传《只是为了好玩》。把村上春树的书除了游记基本上都读完了，王小波所有的书重读了（去年就有读了，今年才读完）。看了很多博客，最值得一提的就是霍炬老师的博客了，从中学到了很多。

# 写作
今年写了一篇短篇小说还发在公众号上了，转到朋友圈算是第一次公开自己写的内容，十月份还写了一篇但是没能完成，然后就是这篇年终总结了。希望自己以后多做一些深度思考然后产出内容，不过也不强制，因为确实重心应该放在别处。

# 赛博朋克
赛博朋克也值得说一下，暑假看到一款主角是猫猫的游戏《Stray》，觉得有趣就买来玩了，一口气从上午打到下午，花了八个小时通关。全部玩下来，最让我感兴趣的不是猫猫元素了，而是游戏的社会背景，猫猫身处在一个赛博朋克世界，某种程度上，我觉得在赛博朋克世界发生的事情现在也在发生着，这一下吸引了我。然后我就一路追溯，看了斯嘉丽约翰逊演的《攻壳机动队》还有动漫，两天看完了《赛博朋克：边缘行者》，等等。下一步还打算看看经典的赛博朋克风格小说像《神经漫游者》这类的。也许有机会会把头发染成蓝色哈哈哈哈。

# 纪录片
其实是通过纪录片认识了两个伟大的人。上半年看霍炬老师的博客知道了Aaron Swartz，某一天心血来潮去看了他的纪录片《The Internet’s own boy(互联网之子)》，然后就被他吸引住了，他是个怀着崇高理想的人，“He is the Internet’s own boy, and the old world killed him”。还有一个人是Eliud Kipchoge基普乔格，非常励志，期中考试那会我就每天看同一个关于他的视频剪辑给自己打鸡血。每一场马拉松，就像抓住一根一根树枝不断往上爬树一样，要一步一步才能获得前所未有的体验，才能不断突破自己。因为自己资质平平，所以忍受痛苦才能学有所成，深以为然。人生是一场马拉松，各种意义上的。

# 活动
下半年作为志愿者参加了开源社举办的线下2022开源年会，玩得很开心，还交到了朋友，不过要说最大的收获，是获得一种协调时间的经历。是这次开源年会是在电磁场与电磁波期中考试的前一天，如果是之前我就不参加了，但我还是没放弃，并且抓紧时间复习电磁场，最后电磁场期中也考的不错。 活动中做的最错误的一件事情是，作为志愿者，我去和一位大概四十多岁的参与方协调活动，前辈一见我就伸手要跟我握手，我有点惊讶，没跟他握手就说起来活动的事情，前辈的手停了一下是自己抽回去的…这是我觉得做的最没礼貌的事情，不会再有下次了 :crying_cat_face: 。 不能放弃，要去参与去经历。

# 尾声
不管怎样，2022年已经结束，2023年即将来临。我坚信自己一定是越来越好的，我会一直保持良善，不断认识自己的缺点并且完善自己。那么希望接下来的一年，可以在专业知识上更进一步，同时心智上要成长得再快一点。下一年的主题应该是考研，希望2023年的12月31号，自己在写年终总结回忆的时候，自己可以满意地长舒一口气。 贺炜解说员在梅西世界杯夺冠赛后这样说道：“任何命运，无论如何复杂漫长，实际上只反映于一个瞬间：人们大彻大悟自己究竟是谁的瞬间”，我要从经历中得到这一瞬间。
