# 2024年第5周周报

本来是昨天要写好发出来，但是因为贪玩就延后到今天了…

# 学业
大部分时间还在做毕设，一步一步前进，有基础知识需要补就去zlib上找电子书看。
申请了几个线上实习，希望可以成功，也把线上实习的信息分享给了朋友。不过线上实习的内容也都和自己的毕设有关系，所以当务之急是做好毕设。

# 社交
这周也在社交方面有了新的理解，意识到自己以前太封闭了，可能就和很多做技术的人想的一样，觉得只要自己技术好就行（虽然现在也不好...），不需要做很多无用社交。但其实越来越意识到合作和交流的重要性，一个人的力量还是太薄弱了。而且不能让自己处于封闭的状态，要把自己打开。另外也不能害怕被拒绝。
意识到自己在社交这方面一直处于不及格的状态，晚上因此失眠，在床上思考良久，那就从现在开始改变罢！

# 生活
每隔几天见到小侄女，都能感受到她的成长。

# 阅读
看完了《埃隆马斯克传》，开始看罗斯的《公司理财》。完全是一本教科书了，似乎还是有的学校的考研用书，所以预计要看很久，不过学经济的朋友推荐说比较好懂。

# 游戏
断断续续在玩lolm，年前的活动奖励还是比较丰厚的。另外自己想更多享受游戏的乐趣又不至于太上瘾，所以在尝试辅助这个位置（原来是AD），玩辅助~~感觉乱玩英雄也无所谓~~可以尝试不同的英雄，还可以游走，不像AD一样只玩那几个英雄，对线期几乎只能待在下路。
