# 2024年第3周周报

还是把周报给捡起来了...记录一下近两周发生的事情吧。

# 旅游

![](../_assets/2024年第3周周报_1770867774404.png)
上周五我们一行六个人开车自驾出发去旅游了，目的地南京。

![](../_assets/2024年第3周周报_1770867775867.png)
周五晚上先到了安徽蚌埠，晚上就在蚌埠住了。我们去找了一家比较有当地特色的饭馆。老板娘大妈听说我们是外地专门来这家店吃的，非常热情，（外向的）姐夫说“跟她一比我们都显得内向”。菜基本都是又咸又辣，可能是当地特色吧，不过很好吃很下饭。晚上我和哥住一个双人间，第一晚还是比较拘谨的，后面就放开了。我们在房间各自打了会游戏就睡觉了。这个哥长得特别帅，和女生连麦玩王者荣耀，我玩英雄联盟手游。
第二天吃完早饭就开车上路了。早饭吃的也是当地比较有特色的鸡汤面。不过跟我们这里不太一样，那天吃的是一碗鸡汤还有一碗面，我把鸡汤全喝完了，面倒是没吃几口。
大概下午一两点到的南京。我们先去酒店放了行李，酒店在新街口附近。然后徒步去姐说的一家比较地道的咸水鸭店，结果好多人在排队，我们就去了附近一家店吃鸭血粉丝汤。我吃完带着小侄女在外面玩，姐夫（小侄女的爸爸）在巷子对面抽烟，小侄女看见准备跑过去，这时候突然来了辆三轮车，三轮车司机好像在看手机，快撞上小侄女的时候他赶紧刹车，我也刚好拽着小侄女，差点撞上小侄女，现在想想我都后怕。
然后我们去了瞻园，看了瞻园的太平天国博物馆和园林，走走路拍拍照就出来了（照片基本都在姐那，我就放点瞎拍的不露家人脸的照片）。

![](../_assets/2024年第3周周报_1770867777167.png)

![](../_assets/2024年第3周周报_1770867779263.png)
出了瞻园我们步行去了夫子庙，人很多很多。去瞻园时出租车司机告诉我们不要在夫子庙买东西，我们就只逛，没有买东西。本来想坐一下秦淮河上的观光船，人太多就放弃了。
第二天去了红山动物园，看见好多动物，不过姐夫觉得不如他们在北京去的野生动物园。中午吃饭的地方食物可能也不新鲜，我和妈还有小侄女当天晚上和第二天都有腹泻症状。在红山动物园转了一天，姐给小侄女买了一个小熊猫玩偶，小侄女拿到特别喜欢，爱不释手。结果三分钟热度，还没出红山动物园就玩腻了，这就是小孩子啊...
第三天去了中山风景区，去逛了寺庙吃了斋饭。斋饭是一碗面加一块素鸡，我觉得味道还不错，不过下午很快就饿了。然后爬了那里的塔，那个塔可能是每层地面不太平的缘故，很陡很吓人，让人不敢靠近塔边缘。一共九层，上去很累，再加上有点恐高，拍了几张照我和妈还有姐就下来了。

![](../_assets/2024年第3周周报_1770867781039.png)

![](../_assets/2024年第3周周报_1770867784099.png)
又去了音乐台，有很多鸽子，还有会随音乐变化的喷泉，人比较多，我们待了一会就走了。然后去了附近的中山陵，那天闭馆，走上去走到馆门口就下来了。今天的旅途就算结束了。

![](../_assets/2024年第3周周报_1770867785994.png)
第四天上午去了一个小博物馆（名字忘了...陈列的是红楼梦和曹雪芹相关的物品），出来去了一家烘焙店（姐做甜品，所以有尝探烘焙店的习惯），然后去吃了蟹黄面。下午去了海底世界，那个海狮还会吓人，游着游着突然吓我一下，把我吓一跳。

![](../_assets/2024年第3周周报_1770867787470.png)

![](../_assets/2024年第3周周报_1770867789463.png)

![](../_assets/2024年第3周周报_1770867795098.png)

![](../_assets/2024年第3周周报_1770867798826.png)

![](../_assets/2024年第3周周报_1770867800901.png)
南京之行就结束了，第二天我们就离开南京了，在合肥待了一晚，姐夫和他叔约了顿饭，我和哥也去了。然后就回家回归正常生活了。
