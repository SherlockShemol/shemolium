# Server, Fedi, Blog, aaand AI


# Server
写这篇博客已经是十一月份的尾巴（11.30），特意看了一下，八月份关掉了自建的Misskey站点回到vnil.de，当时把博客迁到了博客园（没记错的话…）。当时自己的主机到了（妈给买的，谢谢妈），面临的问题是如何在两台电脑间同步博客，之前自己在Obsidian上的笔记都会自动同步到Github,但是`git push`再`git pull`的方法实在繁琐不优雅，所以经过考虑之后换回了Notion。之前只有一台电脑的时候从Notion迁到Obsidian是因为延迟问题受不了，而且上传图片限制大小，但现在面临了新问题（两台电脑的笔记同步），不同的处境考虑更优的解决方案，所以私以为换回Notion是合理的选择。而且使用后发现Notion似乎比以前快了，更可用了，实在是意外之喜。然后我关掉了自己的服务器，一个原因是节省开支，还有一个原因是感觉到服务器是一种负担，似乎想要自己心理上更轻松一些…通过[Nobelium项目](https://github.com/craigary/nobelium)将Notion上的数据表格生成静态博客然后托管到Vercel上（应该是这样）,再绑定上自己的域名，再搬一些原博客上的文章（丢掉了很多文章，有的是因为公式很多，Notion上的数学公式语法和传统markdown不太一样…有的是因为图很多…有的是懒的调整格式…嘛也没关系，再写就是了）,博客就算是迁移好了。其实自己理想的博客就是可以把所有的精力都放到写博客内容上，其它的阻力达到最小（所以之前会有迁到博客园的操作…），现在博客状态就是我理想的状态了。其实这篇文章刚刚关掉服务器迁移好博客的时候就在Notion上创建好page写好标题了，但迟迟没有动笔…今天是另外的契机让我觉得可以把想写的内容都放到一起写一下。

# Fedi
这一段时间使用Fedi很多，随笔就写下自己的想法，或者记录一下日常，看动漫的时候也会截图发到Fedi上，改变了之前的使用Fedi的习惯（开始大量发帖），因为很细碎的事情更频繁的出现在了友们的时间线上打扰友们，不知道友们有没有感到困扰哈哈哈，因此在个人介绍上加了一个“取关请随意”，也谢谢直到现在都没有取关的友们。看动漫的时候真的发了好多帖子。前几天看到关于SNS改变大脑奖励回路的相关内容，感觉到自己其实有点被影响了，因为自己貌似就养成了不定期查看微信和Fedi的习惯。自己其实一直在做一些调整，关掉了微信的朋友圈（虽然今天还没忍住打开朋友圈看了一眼…），卸载了小红书（这个是早就卸载了，现在有需要搜一些聚餐饭店之类的都用网页版，这个做的还可以，没在小红书上浪费时间），希望自己尽量保持，不能让自己的注意力经常被打断。今后尽量克制自己在写代码或学习的时候不打开Fedi。

# Blog
昨天看到Faker不死者的文章实在喜欢，看的是一个虎扑论坛的网页（不太喜欢），突然想到能不能把自己喜欢的文章都在博客上转载一份自己留着看，然后这个想法今天继续扩大，想到其实可以在博客上进行一些内容的“聚合”，比如之前在Fedi上发的看动漫的帖子，完全可以整理成一篇文章，这样也不会打扰Fedi，而且内容上要更完整一些，没有那么碎片化。博客完全是自己的地盘了，也不用担心有没有人看，因为其实整理内容、写文章的过程受益者完全是自己。所以打算把之前的Fedi发的内容（比如看星际牛仔的截图）做个补救，整理一下发在博客上，然后后面可以成文的内容（不细碎的）也都发在博客上。

# AI
应付一些无聊的作业或者写一些无聊的文本的时候，GPT真是帮大忙了。Cursor之前也在我写代码的时候给了我很大的帮助。在遇到一个问题的时候，现在第一反应就是习惯性的问一下GPT，然后再搜索引擎搜索一下。但是我慢慢意识到，在用cursor辅助写代码的时候，我发现我对一些代码的印象开始变得模糊不清了，记得很清楚有天晚上躺在床上，想到白天在cursor帮助下生成的代码，突然想到有个变量好像没有用到是不是可以替代掉，然后第二天检查代码的时候发现变量是被用到的不能被替换的。这样的情况又发生了一两次。我觉得这样不行，就把cursor给删掉了。这几天使用ChatGPT4发现GPT4降智了，而且也没办法读取pdf和图片内容，自己第一反应是找其他GPT帮忙。
我发现，如果自己对GPT生成的内容做修改的话（或者GPT），自己会产生轻微的恐惧，迟迟不肯动手，其实是自己产生了依赖，有惰性了。其实最重要的就是思考的过程和解决问题的思路吧，如果直接让AI给答案，那自己的能力就退化了，更加具有脆弱性了。

所以感觉必须严格限制一下使用AI的场景，水报告水无聊作业的时候是可以用GPT的，GPT就是用来做这个的，然后我们可以节省时间在思考重要的事情上。学习的时候可以借助GPT来帮助自己理解知识，或者让GPT做一些扩展延伸，但是写代码的时候是一定不能使用GPT的，我是要成为优秀程序员的人！

其实依赖翻译软件和依赖AI是一样的道理，自己现在也有在注意，情况改善了很多，发现自己好像读英文内容更快更好了（~~是否是一种错觉~~不是错觉），总之再接再厉。
