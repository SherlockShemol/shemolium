# 年终总结-2024

又到年末，按照惯例该写些年终总结出来，希望自己可以坚持下来写年终总结的习惯。

# 技术
上半年基本上在摸索k8s和kubeedge做自己的毕业设计，毕业设计写完折腾了Misskey站点和博客（七月关了Misskey站然后博客迁到Notion…），五月申请了开源之夏然后中选，暑假在做开源之夏的项目，但是其实在家里效率是有点低的，当时其实是有点害怕每两周的例会，然后开学还再做，发现做着做着就做出来了，回头整理项目又觉得挺简单的这有什么难还做了那么久真笨…其实有时候也想着能有个人可以带一下，而不是自己摸索，大概能进步很快吧，会不会能更快的发现问题。但是自己又觉得这样自己做自己摸索，其实也是学习的过程，虽然慢但自己得到了锻炼，不如就这样看看自己可以走多远。
下半年到十一月可能都在忙OSPP（我太菜了对不起…）…忙完后看了看其他开源项目（volcano等）。shrik3推荐了[Quark Container](https://github.com/QuarkContainer/Quark)开源项目，在社区提issue很快就能得到回应，又看了[LFX的](https://youtu.be/YvC-ytrDEvo)和[CNCF的视频](https://youtu.be/xpMPMt9JEX8)和我觉得还是很不错的。所以学了一些Rust希望尽快为社区有所贡献。
有些事情自己能够看到，知道做这件事情是好的，但是好像需要择时，如果选择做这件事情的时间不对，似乎也会对自己有不利的影响，所以把有些该学的东西往后放一放，尽量让自己的学习成系统化。
明年应该会继续对Sedna贡献,争取成为自己就可以/lgtm的角色（这样省事一点只需要再找一个人就能merge了）。在KubeEdge社区也开了新坑…Quark Container还有想做的东西…eBPF也要学…所以应该不会再拓展其他开源项目了因为已经够多了学不过来了…继续努力吧。

自己其实一直有沉迷进某事的倾向，大一大二的时候，每次想要沉迷的时候我都制止自己，因为我不确定这就是我要走的路，我不敢沉迷进去。现在我觉得可以沉迷了，因为是我选择了这条路。但是我不知道自己还能不能真的沉迷了…

# 生活、社交和情感
记得大概是一月的某个晚上，自己因为社交处于不及格状态而失眠，痛定思痛决定多参加社交活动，此后基本来者不拒，假期去参加了小学同学聚会，去和高中同学打麻将。开学后也和在北京上学的高中联系比较频繁，陪wzy去雍和宫还愿，夜骑长安街，给wzy过生日，打麻将。毕业和舍友一起拍了毕业照，暑假十一个人一起去看了《你的名字》（浩浩荡荡的大部队…）。自己也因为处在人群中而得以稍微清楚自己在人群中的位置，稍微有些清楚哪些事情是否应该合群。上半年还和阿鸟面基了很开心！🐱明年争取多约一些网友面基（有机会的话）。自己还有一个顽疾就是被动心态，在努力克服自己的被动心理。如果说年初社交是不及格状态的话，年末的自己可以打55分（？还是不及格但是及格似乎就在眼前了XD），因为自己还是不会和他人建立联系…

九月和一个以前就认识的朋友联系比较密切，一起吃过几次饭出去玩过两次，在感受到被冷落后选择表白，然后经历了意料之中的失败。失落了一段时间然后精神状态恢复。因为是和自己很不同的人，所以有从对方身上学到一些东西，总之感谢相遇。十月回家向朋友倾诉很多，感谢倾听。自己的情绪自己已经无法处理，所以倾诉给朋友寻求他人帮助消解自己的情绪。
从中学到了什么呢，大概是不需要迎合别人，就做自己就好。
> This above all: to thine own self be true, and it must follow, as the night the day, Thou canst not then be false to any man.
还有不要那么快进入角色…
十一月因为自己不会和他人建立联系而失眠（又失眠？）痛定思痛（又痛定思痛？）决定改变。和他人建立联系（其实就是交朋友…）的能力是需要锻炼的，如果现在不改变那么即使到三十岁情况也不会好转，所以觉得应该去主动接触和一些人交朋友，总之保持真诚和别人交朋友。

自己想要的是那种因为一件件小事而积累起来的信任，现在还是愿意相信这样的东西是存在的。

# 书影音动漫作品
过年和妈看了去看了《热辣滚烫》，陪妈看的，所以妈看的开心就行。
然后上半年还看了咒术回战和芙莉莲，去电影院看了《排球少年：垃圾场决战》，《坂本龙一：杰作》。和wzy去看了《你想活出怎样的人生》。自己在电脑上看了《机器人之梦》。暑假和zsh和dlh看了《只此青绿》（zsh提议的），睡过去大半的时间，这好像是我第一次在电影院睡着，跟中午没午睡有点关系，但是电影本身太无聊是根本原因。
上半年还看了什么因为没有记录都不记得了…下半年看的倒是还专门整理出了一篇文章，基本上就是那些了，然后现在在看06版的fate/stay night,还有很多很多想看的动漫漫画电影！

![](../_assets/年终总结-2024_1770868888377.png)
发现以前似乎不爱听的歌现在能听了，感觉和自己的心更加开放有关系，是很好的事情。十一月开始狂听椎名林檎，一发不可收拾，网易云年度总结的时候连续三年都是黑木渚，但是我知道明年一定不是了，明年会是椎名林檎。

今年没读几本书…《娱乐至死》《闪光的哈萨维》…除此之外似乎没有了，但也问题不大，自己没读理论但有在实践嘛。

# 电子产品
十月份买了主机，是妈给买的，谢谢妈。装了Arch，4070Ti Super的显卡跑Stable Diffusion3。在犹豫要不要在主机上装Windows笔记本装Arch，这样带着电脑回家写代码啥的也方便，Windows还能玩玩游戏，但是换系统装环境实在是太太太太麻烦了，所以应该会作罢（但应该还是会换的…）…

年末的时候用OSPP的奖金购买了懒猫微服，原本设想的是把笔记博客都迁移上去，省下订阅Notion的钱，但是笔记博客迁移起来不是一般的麻烦遂放弃（省钱大失败！）。目前的用途是用它存文件和下载资源做种。现在就是打算迁移应用回点本 :( 表面上看着再光鲜亮丽的东西，上手去摸就能发现瑕疵的。产品本身是很厉害的，虽然不想承认，但是自己确实不是这款产品的目标受众，后悔购买…

# In the end
那么今年大概就到这里啦，希望来年自己可以继续成长，再观察得仔细一点，再深入一点，深入技术，深入人心，深入这个世界。
